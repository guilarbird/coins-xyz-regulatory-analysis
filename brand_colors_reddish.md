# Coins.xyz Brand Guidelines - Reddish/Pink Variant

## Official Color Palette (Reddish Gradient)

Based on brand guidelines page 19, the reddish variant uses a gradient from:
- **Red** (#ED452D) → **Pink/Magenta** → **Purple/Iris** (#7A70FF) → **Blue** (#0062F5)

### Primary Colors for Reddish Variant:
- **Red**: #ED452D (primary accent)
- **Pink/Magenta**: #FF006E (supporting)
- **Purple/Iris**: #7A70FF (supporting)
- **Blue**: #0062F5 (supporting)
- **Cyan/Turquoise**: #00D9FF (supporting)

### Gradient Direction:
Top-left (Red) → Top-right (Pink) → Bottom-right (Purple/Blue) → Bottom-left (Cyan)

### Supporting Neutrals:
- **Black**: #000000 (text)
- **White**: #FFFFFF (backgrounds, text on dark)
- **Gray 50**: #F9FAFB (light backgrounds)
- **Gray 100**: #F3F4F6
- **Gray 200**: #E5E7EB (borders)
- **Gray 800**: #1F2937 (dark text)

## Usage Notes:
- Hero sections: Use full reddish gradient (red → pink → purple → blue)
- Cards/sections: Use single color from palette with appropriate tint
- CTAs: Red #ED452D primary
- Status indicators: Green for success, Red for critical, Blue for info
- Backgrounds: Light gray (#F9FAFB) for contrast against vibrant gradients

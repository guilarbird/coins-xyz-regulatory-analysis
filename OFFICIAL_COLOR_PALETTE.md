# Coins.xyz Official Color Palette

Based on `coinsxyz_brandguidelines_17.pdf`

## Primary Brand Colors (ONLY THESE SHOULD BE USED)

### 1. Iris (Purple)
- **HEX**: `#7A70FF`
- **RGB**: 122, 112, 255
- **CMYK**: 52, 56, 0, 0
- **Usage**: Primary brand color, buttons, accents
- **Tints**: 100, 200, 300, 400, 500, 600, 700, 800, 900

### 2. Green
- **HEX**: `#5DE828`
- **RGB**: 93, 232, 40
- **CMYK**: 60, 0, 83, 9
- **Usage**: Success states, positive indicators, EMI license
- **Tints**: 100, 200, 300, 400, 500, 600, 700, 800, 900

### 3. Blue
- **HEX**: `#0062F5`
- **RGB**: 0, 98, 245
- **CMYK**: 100, 60, 0, 4
- **Usage**: Information, links, VASP license
- **Tints**: 100, 200, 300, 400, 500, 600, 700, 800, 900

### 4. Red
- **HEX**: `#ED452D`
- **RGB**: 237, 68, 45
- **CMYK**: 0, 71, 81, 7
- **Usage**: Alerts, critical items, urgent actions
- **Tints**: 100, 200, 300, 400, 500, 600, 700, 800, 900

### 5. Black
- **HEX**: `#000000`
- **RGB**: 0, 0, 0
- **CMYK**: 0, 0, 0, 0
- **Usage**: Text, dark backgrounds
- **Grays**: 100, 200, 300, 400, 500, 600, 700, 800, 900

### 6. White
- **HEX**: `#FFFFFF`
- **RGB**: 255, 255, 255
- **CMYK**: 0, 0, 0, 0
- **Usage**: Backgrounds, light text on dark
- **Grays**: 50, 100

## ❌ Colors to REMOVE

The following colors are NOT in the official palette and should be replaced:

- **Yellow** (#FEF3C7, #FED7AA, etc.) → Replace with **Iris** or **Green** tints
- **Orange** (#FCA5A5, #F87171, etc.) → Replace with **Red** tints
- **Pink** (#FF69B4, #FF006E) → NOT in official palette, use **Red** or **Iris**
- **Cyan** (#00D9FF) → NOT in official palette, use **Blue**
- **Indigo** (#C7D2FE, #DDD6FE) → Replace with **Iris** tints
- **Generic Grays** (gray-50, gray-100, etc.) → Use **Black** tints only

## Correct Color Mapping

### Status Indicators
- **In Progress**: Iris (#7A70FF) with lighter tints for backgrounds
- **Upcoming**: Blue (#0062F5) with lighter tints for backgrounds
- **Critical/Blocker**: Red (#ED452D) with lighter tints for backgrounds
- **Success/Complete**: Green (#5DE828) with lighter tints for backgrounds

### Licenses
- **VASP License**: Blue (#0062F5)
- **EMI License**: Green (#5DE828)

### Backgrounds
- **Light backgrounds**: White (#FFFFFF) or Black tints (100-200)
- **Card backgrounds**: White with subtle Black tint borders

### Text
- **Primary text**: Black (#000000) or Black-900
- **Secondary text**: Black-600 or Black-700
- **Muted text**: Black-400 or Black-500

## Implementation Rules

1. **NO custom colors** - Only use the 6 official colors above
2. **Use tints** - Each color has 100-900 tints for variations
3. **Gradients** - Only use combinations of official colors
4. **Borders** - Use Black tints (100-300) for subtle borders
5. **Shadows** - Use Black with opacity for shadows

## Gradient Combinations (Official Colors Only)

- **Primary Hero**: Iris → Blue → Green
- **Warm Accent**: Red → Iris
- **Cool Accent**: Blue → Iris
- **Success Flow**: Green → Blue
- **Alert Flow**: Red → Red (darker tint)

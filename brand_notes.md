# Coins.xyz Brand Guidelines - Implementation Notes

## Typography
- **Primary Font**: One Platt (modern, geometric, clean)
- Use for headlines and key numbers
- Pair with sans-serif for body text

## Color Palette (Gradients - GenZ Style)
1. **Iris-Green**: Purple → Blue → Cyan → Green (main hero gradient)
2. **Iris-Red**: Red → Purple → Blue
3. **Iris-Blue**: Blue → Purple → Blue (deep)
4. **Green-Red**: Green → Yellow → Orange → Red
5. **Green-Blue**: Green → Cyan → Blue
6. **Red-Blue**: Red → Purple → Blue → Orange

## Design Principles
- **Vibrant gradients** as backgrounds (not flat colors)
- **Bold typography** with geometric shapes
- **Clean, modern** layouts with ample white space
- **Dynamic elements**: use the "X" symbol from logo as decorative element
- **Light mode** preferred for professional content

## Implementation for Dashboard
- Hero: Iris-Green gradient (purple-to-green)
- VASP sections: Blue tones
- EMI sections: Green tones
- Alerts/Blockers: Red-Blue gradient
- Success/Progress: Green-Blue gradient
- Typography: One Platt for numbers and headlines, Inter for body
